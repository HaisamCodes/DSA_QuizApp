# DSA Quiz App (Qt Based)

### How to Run:
1. Download or Clone this repository.
2. Open **Qt Creator**.
3. Go to `File` -> `Open File or Project`.
4. Select the `CMakeLists.txt` file.
5. Choose your Kit (e.g., Desktop Qt 6.x MinGW).
6. Click **Configure Project**.
7. Press **Run (Ctrl + R)**.

### Features:
- 15+ DSA Questions
- Timer for each question
- Score calculation
